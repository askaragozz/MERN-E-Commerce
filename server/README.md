# MERN E-Commerce

A portfolio-grade e-commerce application built with MongoDB, Express, React, and Node.js.

## Tech Stack
- Frontend: React (Vite/Next TBD), TypeScript, CSS framework Tailwind
- Backend: Node.js, Express, TypeScript, Mongoose
- DB: MongoDB Atlas
- Tooling: ESLint (flat), Prettier, Husky + lint-staged, Test TBD
